# 学习计划

## 总目标

系统掌握推荐系统科研训练所需的基础能力，包括 Python 编程、数学基础、推荐系统核心方法、论文阅读和实验工具使用。

## 阶段安排

### 阶段一：基础补齐

- Python 基础与常用数据处理库
- 线性代数、概率统计和优化基础
- Git、命令行、实验记录工具

### 阶段二：推荐系统入门

- 协同过滤与矩阵分解
- 召回、排序、重排的基本流程
- 常见评估指标与离线实验方法

### 阶段三：论文阅读与复现

- 建立论文阅读模板
- 阅读经典推荐系统论文
- 选择一个小模型进行代码复现

### 阶段四：科研训练产出

- 梳理研究问题
- 设计实验方案
- 汇总实验结果与阶段报告

## 每周复盘

- 本周完成了什么？
- 哪些知识点仍不清楚？
- 有哪些可复用的代码、笔记或实验结论？
- 下周最重要的三个任务是什么？
