# 推荐系统科研训练笔记

这个仓库用于记录推荐系统科研训练过程中的学习计划、每日进展、课程作业和知识沉淀。

## 目录结构

- `learning_plan.md`：整体学习计划与阶段目标
- `daily_logs/`：每日学习记录、问题和复盘
- `assignment/`：课程作业与提交记录
- `knowledge_base/`：长期维护的知识库

## 使用方式

1. 每天在 `daily_logs/` 中记录当天学习内容、产出和问题。
2. 将课程作业放入 `assignment/`，保留解题思路、代码链接和复盘。
3. 把反复出现的概念、方法和工具整理进 `knowledge_base/`。
4. 定期回看 `learning_plan.md`，根据训练进度调整目标。
